# Crop-Disease-Detection

## Overview

A simple ML project implementing federated learning with a quantum-inspired feature transformation for potato leaf disease detection. The model is trained using the publicly available PlantVillage potato dataset(Kaggle) and implemented in Google Colab.

## Features

- Image preprocessing using OpenCV
- Federated learning with multiple clients
- Quantum-inspired feature transformation
- Disease prediction for potato leaves
- Model evaluation using accuracy, F1-score and confusion matrix

## Technologies used

- Python
- Google Colab
- NumPy
- OpenCV
- Scikit-learn
